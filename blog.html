QnA about JavaScript

What is DOM and what is the purpose of DOM?

Full form of DOM is Document Object Model. Developer can modify Document structure, styles and content using DOM.

How will you select HTML elements using DOM? Name of the DOM methods.

There are some methods to select HTML elements using DOM. We can use getElementById(), getElementsByClassName(), querySelector(), querySelectorAll() to select HTML elements.

What is event bubble?

Event bubble is a way of propagation. By event bubble the inner most element will handled frist and then the outer elements.

What is a callback function and why will you use it?

A function which is passed as another functions parameter. Callback function is called inside another function to do partial work.
